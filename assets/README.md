# Sample image placeholder

This directory contains sample assets used in the demo notebook and for testing.

## How to add a sample image

Place a file named `sample_image.jpg` in this directory.
You can download one using:

```bash
curl -L "https://ultralytics.com/images/bus.jpg" -o assets/sample_image.jpg
```

Or use any `.jpg` / `.png` image of your choice.
